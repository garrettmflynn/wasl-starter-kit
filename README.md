# wasl-starter-kit 
[![License: AGPL v3](https://img.shields.io/badge/license-AGPL_v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)
[![Discord](https://img.shields.io/badge/community-discord-7289da.svg?sanitize=true)](https://discord.gg/CDxskSh9ZB)

**wasl-starter-kit** will get you started building a WASL application from scratch. It comes with an HTML file with boilerplate WASL code to compile your application. Edit the `index.wasl.json` file and related Components in the `app` folder to change its behavior!

## Getting Started
Make sure to host all files alongside the index.html file. We recommend using the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension on Visual Studio Code.

Additional Components to use for your projects can be found in our [official Component repository](https://github.com/brainsatplay/components).

## Support
If you have questions about developing with the [WASL Starter Kit](https://github.com/brainsatplay/wasl-starter-kit), feel free to start a conversation on [Discord](https://discord.gg/tQ8P79tw8j) or reach out directly to our team at [contact@brainsatplay.com](mailto:contact@brainsatplay.com).
